Nosso Compromisso com a Acessibilidade

A ONG Anajo está comprometida em garantir que nosso site seja acessível a todos os usuários, independentemente de suas habilidades ou deficiências. Trabalhamos continuamente para melhorar a usabilidade e acessibilidade de nosso site seguindo as diretrizes WCAG 2.1 (Web Content Accessibility Guidelines) nível AA.

Recursos de Acessibilidade Disponíveis

  • Ajuste de Tamanho de Texto: Você pode aumentar ou diminuir o tamanho do texto em todo o site usando os botões A+ e A- na barra de acessibilidade no topo da página.
  • Modo de Alto Contraste: Ative o modo de alto contraste para melhorar a legibilidade clicando no botão "Contraste" na barra de acessibilidade.
  • Navegação por Teclado: Todo o site pode ser navegado usando apenas o teclado. Use a tecla Tab para navegar entre elementos e Enter para ativar links e botões.
  • Link para Pular Navegação: Um link invisível "Pular para o conteúdo principal" aparece quando você pressiona a tecla Tab pela primeira vez, permitindo que você pule diretamente para o conteúdo principal da página.
  • Estrutura Semântica: Nosso site utiliza marcação HTML semântica adequada, com cabeçalhos hierárquicos e landmarks ARIA para facilitar a navegação com leitores de tela.
  • Textos Alternativos: Todas as imagens informativas possuem textos alternativos descritivos para usuários de leitores de tela.

Como Utilizar os Recursos de Acessibilidade

  1. Localize a barra de acessibilidade no topo da página
  2. Clique no botão A+ para aumentar o tamanho do texto
  3. Clique no botão A- para diminuir o tamanho do texto
  4. Sua preferência será salva para futuras visitas ao site

  1. Localize a barra de acessibilidade no topo da página
  2. Clique no botão Contraste
  3. O site mudará para cores de alto contraste
  4. Clique novamente para retornar ao modo normal
  5. Sua preferência será salva para futuras visitas ao site

  • Tab: Navega para o próximo elemento interativo
  • Shift + Tab: Navega para o elemento interativo anterior
  • Enter: Ativa o elemento selecionado (links, botões)
  • Espaço: Ativa botões e controles de formulário
  • Setas: Navega dentro de menus e listas

Tecnologias Assistivas Compatíveis

Nosso site foi testado e é compatível com as seguintes tecnologias assistivas:

  • Leitores de tela (NVDA, JAWS, VoiceOver)
  • Software de ampliação de tela
  • Navegação por teclado
  • Reconhecimento de voz

Limitações Conhecidas

Estamos trabalhando continuamente para melhorar a acessibilidade do nosso site. Atualmente, estamos cientes das seguintes limitações:

  • Alguns documentos PDF mais antigos podem não ser totalmente acessíveis
  • Alguns conteúdos de terceiros incorporados podem ter limitações de acessibilidade
